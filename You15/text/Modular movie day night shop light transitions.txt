Modular movie day night shop light transitions

To create a smooth visual transition when the mall shifts from day to night, your script needs to handle three distinct lighting phases: Business Hours (Day), Closing Time (Transition), and After Hours (Night).
Because your camera is completely flat and perpendicular to the walls, you can achieve a highly realistic lighting shift using Image Mask Blending. This technique avoids harsh, sudden cuts and creates a smooth, atmospheric sunset and shutdown sequence.

1. The Three Lighting Assets Needed
Instead of using a single background image, your artist will deliver two distinct background sheets and one lighting map designed to fit your real-world dimensions:

	1.	mall_structure_day.png – Bright, ambient mall lighting with active interior shop illumination.
	2.	mall_structure_night.png – The same scene, but with the main ceiling grids turned off. Only overhead safety running lights and illuminated emergency exits remain active.
	3.	guard_flashlight_mask.png – A 1920x1080 transparent sheet containing a soft, semi-transparent white cone gradient. This represents the light beam cast from the guard’s hand.

2. The Timeline Trigger Logic
Using your 6-hour video timeline (648,000 frames total), we can divide the day/night cycle into four distinct operational shifts:

	•	Frames 0 to 324,000 (Hours 0–3): Day Shift. The mall is bright and open. Storefront advertisements are fully illuminated.
	•	Frames 324,000 to 432,000 (Hours 3–4): Closing Time. The sun begins to set. The script slowly dissolves the day background into the night background over the course of an hour.
	•	Frames 432,000 to 540,000 (Hours 4–5): Deep Night. The mall is closed and dark. The guard turns on his flashlight to illuminate the corridors.
	•	Frames 540,000 to 648,000 (Hours 5–6): Morning Sunrise. The dawn light slowly blends the bright daytime assets back into the scene.

3. Python Implementation: The Smooth Transition Engine
This logic replaces the basic image loading section inside your main loop. It calculates a frame-by-frame alpha blend value (blend_alpha) to seamlessly transition between the daytime and nighttime artwork.
# --- ADVANCED DAY/NIGHT LIGHTING TRANSITION ENGINE ---

# Establish transition frame benchmarks based on the 6-hour timeline
CROSSFADE_DURATION = 1 * 60 * 60 * FRAME_RATE # 1 Hour transition phase (108,000 frames)
CLOSING_START = 3 * 60 * 60 * FRAME_RATE      # Triggers at Hour 3
SUNRISE_START = 5 * 60 * 60 * FRAME_RATE      # Triggers at Hour 5

# Initialize the canvas selection for this specific frame
if CLOSING_START <= cycle_frame < (CLOSING_START + CROSSFADE_DURATION):
    # PHASE 1: CLOSING TIME CROSSFADE (Day dissolves into Night)
    # Calculate a value shifting from 0.0 (Pure Day) to 1.0 (Pure Night)
    progress = (cycle_frame - CLOSING_START) / CROSSFADE_DURATION
    canvas_base = Image.blend(bg_day, bg_night, alpha=progress)
    is_mall_open = True if progress < 0.5 else False # Shut down ads halfway through sunset

elif SUNRISE_START <= cycle_frame < (SUNRISE_START + CROSSFADE_DURATION):
    # PHASE 2: SUNRISE CROSSFADE (Night dissolves into Day)
    # Calculate a value shifting from 0.0 (Pure Night) to 1.0 (Pure Day)
    progress = (cycle_frame - SUNRISE_START) / CROSSFADE_DURATION
    canvas_base = Image.blend(bg_night, bg_day, alpha=progress)
    is_mall_open = True if progress > 0.5 else False

elif (CLOSING_START + CROSSFADE_DURATION) <= cycle_frame < SUNRISE_START:
    # PHASE 3: DEEP NIGHT OPERATIONS
    canvas_base = bg_night.copy()
    is_mall_open = False
else:
    # PHASE 4: STANDARD BRIGHT BUSINESS HOURS
    canvas_base = bg_day.copy()
    is_mall_open = True

4. Dynamic Ad and Flashlight Integration
To make sure your advertisers' storefronts don't look awkwardly bright in the middle of a dark, closed mall, use your is_mall_open status variable to change how the ads are blended into the scene:
# Fetch active brand PNG files from your spreadsheet pipeline
ad_1, ad_2 = get_live_ads()

if not is_mall_open:
    # If the mall is closed, apply a heavy desaturation and brightness drop to the ads
    # This simulates the store's interior display window lights clicking shut
    ad_1 = ImageEnhance.Brightness(ad_1).enhance(0.15)
    ad_1 = ImageEnhance.Color(ad_1).enhance(0.3)
    ad_2 = ImageEnhance.Brightness(ad_2).enhance(0.15)
    ad_2 = ImageEnhance.Color(ad_2).enhance(0.3)

# Stack the modified advertiser assets into their designated wall slots
canvas_base.alpha_composite(ad_1, dest=(350, 500))
canvas_base.alpha_composite(ad_2, dest=(1120, 500))

# Overlap your security guard sprite
canvas_base.alpha_composite(sprite, dest=(guard_x, guard_y))

# PHASE 5: FLOATING FLASHLIGHT ILLUMINATION (Night Only)
if not is_mall_open:
    # Calculate the exact pixel offset so the flashlight beam tracks with the guard's hands
    # If the guard walks right, project the light cone forward; if left, invert the mask
    flashlight_offset_x = guard_x + (60 if direction == "right" else -250)
    flashlight_offset_y = guard_y + 60 # Hand height level
    
    # Composite the light beam overlay onto the dark scene
    canvas_base.alpha_composite(guard_flashlight_mask, dest=(flashlight_offset_x, flashlight_offset_y))

# Finish the frame by overlaying your foreground structural pillars
canvas_base.alpha_composite(foreground, dest=(0, 0))

Why This Technique is Highly Effective

	•	Seamless Brand Protection: Your real-world advertisers remain clearly visible during the day shift when viewer traffic is high. During the night cycle, their logos fade naturally into the background shadows, preserving the relaxing, isolated aesthetic of the ambient film.
	•	Low System Overhead: Because Image.blend() is executed natively inside Pillow using optimized C-libraries, crossfading two massive 1080p frames takes less than 2 milliseconds. This allows you to generate smooth light transitions without lagging your live stream feed.
