Step 3: Run Your Pre-Flight System Diagnostics
Open your computer's command line or terminal inside the Mall_Patrol_Network directory and run your script directly using Python:
python3 stream_engine.py

	•	What to look for: The script should output [SUCCESS] All core assets validated. System initialized. Starting infinite loop... to the screen.
	•	It will immediately begin printing raw, rapid binary code characters to your terminal window. This is completely correct—it means your Python script is successfully piping the uncompressed frame data directly to your computer's standard output RAM stream instead of cluttering your hard drive.
	•	Press CTRL + C on your keyboard to stop the script.
